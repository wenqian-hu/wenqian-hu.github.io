Send Love IoT | How to make long-distance communication tangible?

Time 1 week | 2017
Toolkit Arduino Yún, API
Course Developing Web
Context Interactive Media Arts, New York University Shanghai

Project Description

Send Love is an Internet of Things project designed for long distance relationship lovers. Transferring emotions into tangible and physical product, this projects seeks to an alternative way of distant communication other than on-screen.

Ideation

When pressing the “send some love” button from the app, the heart installed in the physical box will light up. Accordingly, when pressing the physical button as a response, the application will receive a notification as a reminder of love.

It also includes weather API and can visualize the weather condition in the city of your significant other.
