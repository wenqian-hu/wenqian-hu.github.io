Project Name // Bralette as Statement

Toolkit // Arduino, neopixel, buzzer, pressure sensor, flex sensor

Instructor // Jingwen Zhu

Location // Interactive Telecommunication Program, New York University


Project Description

Bralette As Statement is an interative installation / wearable that draws attention to female sexual harrasment. This project is tangible, silent, while using wires and blinking morse code as self-defense.

Ideation

This is a wearable against sexual harassment. When someone touches or grabes the bralette, the neopixels will blink the morse code of "HELP ME".

The white silky fabrics represents the innocence of victims. Wires are made into a steel ring outside the bralette, meaning what is used to constrain the female body becomes a weapon. Victim blaming sentences are written all over the bralette, creating a sharp contrast to the silence of victims.

Process
